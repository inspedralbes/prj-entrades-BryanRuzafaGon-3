# Teatre Regalia - Plataforma de Venda d'Entrades Real-Time

Aquest projecte ha estat desenvolupat com a Treball de Recerca (TR) per al segon curs de Desenvolupament d'Aplicacions Web (DAW).

- **Desenvolupador:** Bryan Ruzafa Gonçalves
- **Centre:** Institut Pedralbes
- **Curs:** 2DAW 2025-2026
-
## 🎯 Objectiu del Projecte
L'objectiu principal és implementar un sistema de reserva i venda d'entrades que gestioni la concurrència massiva en temps real. Mitjançant WebSockets (Socket.IO), el sistema garanteix que cap seient pugui ser reservat o venut per dos usuaris simultàniament, oferint una experiència immersiva de cinema de luxe.

## 🚀 Estat del Projecte
- **Estat:** 100% Finalitzat i Funcional.
- **Backend:** Laravel 11 (API REST).
- **Temps Real:** Node.js + Socket.IO (Server independent).
- **Frontend:** Nuxt 4 + Pinia (Composition API).
